# Configurable Dashboard

A simple portfolio-style dashboard built with vanilla HTML, CSS, and JavaScript.

## Features

- Responsive layout for desktop, tablet, and mobile
- Collapsible sidebar navigation
- Dark/light theme toggle
- Accessible content structure and navigation
- Clean modular CSS organization

## Project Structure

- index.html — Main page markup
- css/ — Stylesheets
  - base/ — Reset, typography, and theme variables
  - components/ — Reusable UI components
  - layout/ — Page and sidebar layout styles
- js/ — JavaScript behavior
  - main.js — App initialization
  - sidebar.js — Sidebar toggle and navigation
  - theme.js — Theme switching logic

## Preview Locally

Open the project folder in a browser, or run a simple local server from the project root:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000/
```

## Notes

This project is designed as a lightweight front-end portfolio/dashboard demo and does not require any build tools or dependencies.
